# README Usuario

Este proyecto se llama `generador de equipos` y sirve para crear un equipo de jugadores segun sus habilidades. Cargas los nombres, defensa y ataque, y despues elegis como queres formar el equipo. El sistema elige automaticamente al arquero y reparte jugadores entre defensa, medio y delantera.

Tambien permite ver jugadores cargados, cambiar nombres, editar estadisticas y trabajar con distintas versiones del proyecto.

## Carpetas

`versiones/`

Carpeta donde estan las versiones principales, previews, betas y la subcarpeta `version final`.

`versiones/version final/`

Carpeta donde esta el archivo jugable final `generador de jugadores.py`.

`versiones/versiones preliminares(cambios muy pequeños)/`

Carpeta para guardar cambios muy pequeños antes de convertirlos en una preview o version estable.

`crash_reports/`

Carpeta con reportes o archivos relacionados con errores criticos.

`jugables/`

Carpeta donde se guardan los `pycache` movidos desde las versiones.
