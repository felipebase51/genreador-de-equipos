# README Usuario - Generador de Equipos

## Que es

Un programa en Python que te deja cargar jugadores con stats de defensa y ataque, y arma un equipo solo. Elige arquero, reparte defensas, medios y delanteros segun tus indicaciones.

## Como se usa

```
python "versiones/version final/generador de jugadores.py"
```

1. Decis cuantos jugadores vas a cargar.
2. Ingresas nombre, defensa y ataque de cada uno.
3. Escribis `crear` y elegis cuantos defensas, medios y delanteros queres.
4. El programa muestra la formacion armada.

### Comandos

| Comando | Que hace |
|---|---|
| `crear` | Arma el equipo |
| `editar [nombre]` | Cambia defensa o ataque |
| `editname [nombre]` | Cambia el nombre |
| `print` | Muestra la lista |
| `resetear si` | Vuelve a empezar |

## Donde esta cada cosa

- `versiones/version final/generador de jugadores.py` - el archivo para ejecutar
- `versiones/` - todas las versiones desde beta hasta w-5
- `CHANGELOG.md` - historial completo
- `actualizar_version_final.py` - script para mantener la version final al dia

## Requisitos

- Python 3 (probado en 3.14)
- En Koodland: maximo 6 jugadores
- En Windows: sin limite

Para mas info tecnica, mira `README.md`.
