# README Usuario

Este proyecto se llama `generador de equipos`. Cargas jugadores con nombre, defensa y ataque, y el programa arma el equipo solo.

Viene con todas las versiones desde la beta hasta la version final. El archivo jugable esta en `versiones/version final/generador de jugadores.py`.

Para mas detalles, mira `README.md` (el que se ve en GitHub).
